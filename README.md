# Atlas
Atlas Desk Series

A matching set of minimalist desk accessories, designed and modeled as a unified system rather than as standalone products. Each component shares a consistent design language clean lines, a unified material and finish, and shared branding so the pieces read as one cohesive system on a desk rather than a collection of separate accessories.


Components

Laptop Stand

Elevates a laptop to a more ergonomic typing and viewing angle while improving airflow beneath the chassis.

Vertical Laptop Stand

A space-saving variant of the laptop stand that holds a laptop upright, reducing desk footprint when horizontal space is limited.

Atlas Float

A floating phone stand with integrated Qi wireless charging. Suspends the phone above the desk surface while charging it wirelessly, eliminating the need for a separate charging puck.

Stationery Holder

An upright organizer for pens, post-it notes, and other small desk items.

Tray

A general-purpose tray for loose items, small parts, and drawer storage.
